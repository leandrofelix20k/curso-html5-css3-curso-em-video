Links

Valores de atributos

  • _blank: abre o link em uma nova janela.
  • _self: Abre o documento vinculado no mesmo quadro
  • _top: abre o documento vinculado em todo o corpo da janela
  • _parent: Similar ao _top em uma referência a janela mãe
    • .

Natureza dos links

  • next - indica que o link é para a próxima parte do documento
  • prev - indica que o link é para a parte anterior do documento atual
  • author - indica que é um link para o site do autor do artigo atual
  • external - indica que é um link para outro site que não faz parte do site

    • Exemplos:

    _blank

    Acesse o Meu repositório público no GitHub

    _self (padrão)

    Acesse o Meu repositório público no GitHub

    _top

    Acesse o Meu repositório público no GitHub

    _parent

    Acesse o Meu repositório público no GitHub
